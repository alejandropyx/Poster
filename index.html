Using physics priors to characterize and predict dynamics on videos

Alejandro Castaneda, Nergis Tomen, Daan Brinks and Jan Van Gemert

Introduction

Nowadays, technology allows us to record at high speeds different physical phenomena, like free fall, the movement of a sprint, the trajectory of the planets, and the activity of the neurons. Scientists have modeled these dynamics into equations used daily to make predictions. Video predictions are a hot topic in computer vision since they help track systems, decision-making, etc. However, when those videos present physical behavior, we don’t know how to integrate this knowledge to improve and generalize our predictions. Therefore in this project, we aim to study how we can induce physical priors into the model to characterize the event and make a video prediction

Study case

To approach this problem, we set the microscopy problem to identify the dynamic of the brain's neurons for high-temporal resolution videos.

Neuron dynamics on time
Neurons activity has been widely studied to characterize and find the inner mechanisms of the cell that explain the common patterns they exhibit. In particular, the Hodgkin & Huxley model is one of the most popular models since it accurately describes the interaction of the cell’s ion channels. This relation is described in the following differential equation:

$$ {I = C_m \frac{dV_m}{dt} + g_K (V_m - V_K) + g_{Na} (V_m - V_{Na}) + g_l (V_m - V_l) } $$

So we would like to extract the parameters \(g_n\) that describe the dynamics from the video and then use them for video prediction. Therefore we can find helpful information for neuroscientists and study the evolution of the brain's neuron networks.

Methods

Architecture

To achieve our goal, we propose to use an autoencoder architecture with an intermediate step that consists of the physical model to make the prediction.


Encoder: It is trained to predict the time-dependent parameters of the equation.

Predictor: is our mathematical model. It contains learnable parameters that correspond to the constant parameters of our equation.

Decoder: will be trained to transform the time-dependent values to recover the image that represents the corresponding dynamics


Loss Function

In the proposed method, the cost function plays a key role since it needs to optimize the reconstruction but also needs to validate the predicted dynamics, as follows:


L1: Validates the encoded input can be reconstructed.
L2: Validates the predicted state can be reconstructed.
L3: Makes sure the predicted space satisfies the imposed physics.
L4: Allow that the prediction of the dynamics in different times is consisted with the model
$$ { Loss = \alpha (L1+L2) + \beta (L3) + \beta (L4) } $$
